# Project Purpose

**Cumbre Bachilleres UAQ** is a Next.js web application project.

- **Name**: cumbre-bachilleres
- **Type**: Web application (Next.js)
- **Description**: A web platform for "Cumbre | Bachilleres UAQ" (likely an event or program for bachilleres/high school students at UAQ - Universidad Autónoma de Querétaro)
- **Status**: Initial development phase (v0.1.0)

The project is built with modern React/Next.js stack and uses Tailwind CSS v4 for styling.
